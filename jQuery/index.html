Basic Flow of a jQuery Application

Most jQuery applications will follow a sequence of steps like:

Locate one or more page elements according to some criteria.
Perform some action(s) on those elements: like Modify CSS, Modify HTML, Add event handlers Etc.

jQuery ready() Method

    A page can't be manipulated safely until the document is "ready." jQuery detects this state of readiness for you. Code included inside $(document).ready() will only run once the page Document Object Model (DOM) is ready for JavaScript code to execute. Code included inside $(window).on("load",function() { ... }) will run once the entire page (images or iframes), not just the DOM, is ready.

    <script type="text/javascript"> // A $(document).ready() block. $(document).ready(function() { console.log("ready!"); }); </script>

    We can write $(document).ready() in shorthand $() form .

    <script type="text/javascript"> // Shorthand for $(document).ready() $(function() { console.log("ready!"); }); </script>

    You can also pass a named function to $(document).ready() instead of passing an anonymous function.

    <script type="text/javascript"> $(document).ready(mycustomFn); // or: $(window).on( "load", mycustomFn ); // Below is the function which is to be called in ready function instead of an anonymous function. function mycustomFn(jQuery) { // Code to run when the document is ready. } </script>

    The example below shows $(document).ready() and $(window).on("load") in action. The code tries to load a website URL in an <iframe> and checks for both events:

    Example

    Try this code »
    <html> <head> <script src="https://code.jquery.com/jquery-1.9.1.min.js"></script> <script> $(document).ready(function() { console.log("document loaded"); }); $(window).on("load", function() { console.log( "window loaded"); }); </script> </head> <body> <iframe src="http://techcrunch.com"></iframe> </body> </html>

jQuery - AJAX

    AJAX is used for exchanging data with a server, and updating parts of a web page - without reloading the whole page. Using jQuery AJAX methods we can request text, HTML, XML, or JSON from a remote server using both HTTP Get and HTTP Post methods and you can load the external data directly into the selected HTML elements of our web page

    • jQuery load() Method
    • jQuery $.get() Method
    • jQuery $.post() Method

    The jQuery load() method is a simple, but powerful AJAX method. The load() method loads data from a server and puts the returned data into the selected element. The following example loads the content of the file "testdata.txt" into a specific <div> element:

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("button").click(function(){ $("#maindiv").load("testdata.txt"); }); }); </script>

    The $.get() method requests data from the server with an HTTP GET request.

    Syntax:  $.get(URL,callback);

    The required URL parameter specifies the URL you wish to request.

    The optional callback parameter is the name of a function to be executed if the request succeeds.

    The following example uses the $.get() method to retrieve data from a file on the server:

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("button").click(function(){ $.get("home.aspx", function(data, status){ alert("Data: " + data + "\nStatus: " + status); }); }); }); </script>

    The $.post() method requests data from the server using an HTTP POST request.

    Syntax:  $.post(URL,data,callback);

    The required URL parameter specifies the URL you wish to request.

    The optional data parameter specifies some data to send along with the request.

    The following example uses the $.post() method to send some data along with the request:

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("button").click(function(){ $.post("submitPage_post.aspx", { name: "Ramesh", city: "Hyderabad" }, function(data,status){ alert("Data: " + data + "\nStatus: " + status); }); }); }); </script>

jQuery - Selectors

    There are several selectors available for jQuery But All the jQuery selectors are based on the same principle except some tweaking. we have listed some important selectors that are used in most frequently

    • Tag ID Selector
    • Tag Class Selector
    • Tag Name Selector
    • Attribute Selector
    • Compound Selector
    • Custom Selector

    The Tag ID selector selects a single element with the unique ID on the page. For example, the following jQuery code will select and highlight an element having the ID attribute id="elementid", when the document is ready.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("#elementid").css("background", "yellow"); }); </script>

    The Tag Class selector selects the elements with a specific class Name in Document. For example, the following jQuery code will select and highlight the elements having the class attribute class="highlight" when the document is ready.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $(".highlight").css("background", "yellow"); }); </script>

    The Tag name selector selects the elements based on the element name in the DOM. For example,the following jQuery code will select and highlight all the tables when the document is ready.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("table").css("background", "yellow"); }); </script>

    The Tag attribute selector selects an element by one of its HTML attributes, such as a link's href attribute or an input's type attribute, etc. For example, the following jQuery code will select and highlight all the text inputs i.e. textbox elements with the type="text", when the document is ready.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ // Highlight paragraph elements $('input[type="text"]').css("background", "yellow"); }); </script>

    Compound selector means combining the multiple selectors to make your selection even more precise. For instance, you can combine the class selector with an element selector to find the elements in a document that has certain type and class.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ // Highlight only paragraph elements with class mark $("p.mark").css("background", "yellow"); // Highlight only span elements inside the element with ID mark $("#mark span").css("background", "yellow"); // Highlight li elements inside the ul elements $("ul li").css("background", "red"); // Highlight li elements only inside the ul element with id mark $("ul#mark li").css("background", "yellow"); // Highlight li elements inside all the ul element with class mark $("ul.mark li").css("background", "green"); // Highlight all anchor elements with target blank $('a[target="_blank"]').css("background", "yellow"); }); </script>

    jQuery provides its own custom selectors to further enhancing the capabilities of selecting elements on a page.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ // Highlight table rows appearing at odd places $("tr:odd").css("background", "yellow"); // Highlight table rows appearing at even places $("tr:even").css("background", "orange"); // Highlight first paragraph element $("p:first").css("background", "red"); // Highlight last paragraph element $("p:last").css("background", "green"); // Highlight all input elements with type text inside a form $("form :text").css("background", "purple"); // Highlight all input elements with type password inside a form $("form :password").css("background", "blue"); // Highlight all input elements with type submit inside a form $("form :submit").css("background", "violet"); }); </script>

jQuery Filtering

    jQuery provides several methods such as filter(), first(), last(), eq(), slice(), has(), not(), etc. that you can use to narrow down the search for elements in a DOM tree.

    • first() Method
    • last() Method
    • eq() Method
    • filter() Method
    • not() Method
    • has() Method
    • slice() Method

    A jQuery object that represents a set of DOM elements, the .first() method constructs a new jQuery object from the first element in that set.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("ul li").first().addClass("highlight"); }); </script>

    A jQuery object that represents a set of DOM elements, the .last() method constructs a new jQuery object from the last element in that set.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("ul li").last().addClass("highlight"); }); </script>

    The jQuery eq() method filters the set of matched elements and returns only one element with a specified index number. The following example will highlight the second <li> element within the <ul> element by adding the class .highlight on document ready.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("ul li").eq(2).addClass("highlight"); }); </script>

    The jQuery filter() method can take the selector or a function as its argument to filters the set of matched elements based on a specific criteria. The supplied selector or function to the filter() method is tested against each element in the set of matched elements and all the elements that matching the supplied selector or pass the function's test will be included in the result.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("ul li").filter(":even").addClass("highlight"); }); </script>

    The jQuery not() method filters the set of matched elements and returns all elements that does not met the specified conditions.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("ul li").not(":even").addClass("highlight"); }); </script>

    The jQuery has() method filters the set of matched elements and returns only those elements that has the specified descendant element. The following example will highlight all the <li> elements that has the descendant code><ul> elements.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("ul li").has("ul").addClass("highlight"); }); </script>

    The jQuery slice() method filters the set of matched elements specified by a range of indices. This method accepts start and end (optional) index number as arguments, where the start index specifies the position at which the elements begin to be selected and the end index specify the position at which the elements stop being selected.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("ul li").slice(0, 4).addClass("highlight"); }); </script>

Commonly Used jQuery Methods

    Events are actions that can be detected by your Web Application. mouse click, web page loading, Taking mouse over an element, Submitting an HTML form, A keystroke on your keyboard are some examples for events. Methods are attached to the events we have listed some important jQuery methods that are used regularly in a web application development.

    • click()
    • dblclick()
    • hover()
    • mouseenter() and mouseleave()
    • keypress()
    • keydown()
    • keyup()
    • change()
    • focus()
    • blur()

    The click() method attaches an event handler function to an HTML element. The function is executed when the user clicks on the HTML element.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("p").click(function(){ $("p").fadeOut(1000); alert('Click Event Works'); }); }); </script>

    The dblclick() method attaches an event handler function to an HTML element. The function is executed when the user double clicks on the HTML element.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("p").dblclick(function(){ $("p").fadeOut(1000); alert('Double Click Event Works'); }); }); </script>

    The hover() method specifies two functions to run when the mouse pointer hovers over the selected elements. This method triggers both the mouseenter and mouseleave events.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $("p").hover(function(){ $("p").slideUp(1000); alert('Hover Event Works'); }); }); </script>

    The mouseenter() method attaches an event handler function to an HTML element. The function is executed when the mouse pointer enters the HTML element: The mouseleave() method attaches an event handler function to an HTML element. The function is executed when the mouse pointer leaves the HTML element:

    Example

    Try this code »
    <script> $(document).ready(function(){ $('img').mouseenter(function(){ $('img').attr({'src':'assets/images/4.jpg'}); }); $('img').mouseleave(function(){ $('img').attr({'src':'assets/images/6.jpg'}); }); }); </script>

    The keypress() method triggers the keypress event, or attaches a function to run when a keypress event occurs.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $('input').keypress(function(){ $("p").show().fadeOut(); }); }); </script>

    The keydown() method triggers the keydown event, or attaches a function to run when a keydown event occurs.

    Example

    Try this code »
    <script type="text/javascript"> <script type="text/javascript"> $(document).ready(function(){ $('input').keydown(function(){ $("p").show().fadeOut(); }); }); </script>

    The keyup event occurs when a keyboard key is released. The keyup() method triggers the keyup event, or attaches a function to run when a keyup event occurs.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $('input').keyup(function(){ $("p").show().fadeOut(2000); }); }); </script>

    The change event occurs when the value of an element has been changed (only works on <input>, <textarea> and <select> elements). The change() method triggers the change event, or attaches a function to run when a change event occurs.

    Example

    Try this code »
    <script type="text/javascript"> $("select").change(function () { document.getElementById("Result").innerHTML="Selected Technology: "+document.getElementById("selectedvalue").value; }); </script>

    The focus event occurs when an element gets focus (when selected by a mouse click or by "tab-navigating" to it). The focus() method triggers the focus event, or attaches a function to run when a focus event occurs.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $('input').focus(function(){ $("p").show().fadeOut(2000); }); }); </script>

    The blur event occurs when an element loses focus. The blur() method triggers the blur event, or attaches a function to run when a blur event occurs.

    Example

    Try this code »
    <script type="text/javascript"> $(document).ready(function(){ $('input').blur(function(){ $("p").show().fadeOut(2000); }); }); </script>
