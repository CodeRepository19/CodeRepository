Hide/Show Password Example:

Source Code:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"/> <meta name="viewport" content="width=device-width, initial-scale=1"/> <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.2.1/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js" > </script> <title>Hide/Show Password</title> <script> // Change the type of input to password or text function Toggle() { var temp = document.getElementById("password"); if (temp.type === "password") { temp.type = "text"; } else { temp.type = "password"; } } </script> </head> <body> <b><p>Click on the checkbox to show or hide password: </p></b> <b>Password</b>: <input type="password" value="123456789" id="password"> <input type="checkbox" onclick="Toggle()"> <b>Show Password</b> </body> </html>
